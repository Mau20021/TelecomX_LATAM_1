📊 Challenge Telecom X LATAM – Análisis de Cancelación de Clientes (Churn).

📌 Descripción del Proyecto
Análisis de datos realizado sobre la base de clientes de Telecom X LATAM con el objetivo de identificar los principales factores que influyen en la cancelación del servicio (churn) y generar recomendaciones estratégicas para mejorar la retención.

Se evaluaron variables como tipo de contrato, antigüedad del cliente, cargos mensuales, servicios adicionales contratados y comportamiento general de los usuarios.

🎯 Objetivo

Identificar los factores que incrementan la probabilidad de cancelación mediante un análisis exploratorio de datos y presentar conclusiones basadas en evidencia cuantitativa.

🛠 Herramientas Utilizadas

Python
Pandas
NumPy
Matplotlib / Seaborn
Jupyter Notebook

📊 Principales Hallazgos

Los clientes con contrato mensual presentan mayor tasa de cancelación.
La baja antigüedad está directamente relacionada con mayor churn.
Cargos mensuales más altos influyen en la decisión de cancelación.
Clientes con menos servicios adicionales muestran mayor probabilidad de abandono.

🧠 Conclusión

Con base en el análisis realizado, la cancelación de clientes está principalmente asociada a contratos de corto plazo y baja fidelización. Se recomienda fortalecer estrategias de retención temprana, incentivar contratos a largo plazo y ofrecer beneficios adicionales para reducir la tasa de churn.
