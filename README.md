📊 Telecom X – Análisis de Cancelación de Clientes (Churn)
📌 Descripción

Este proyecto analiza datos de cancelación de clientes (churn) de Telecom X LATAM con el objetivo de identificar los principales factores que influyen en la pérdida de clientes y generar insights que apoyen estrategias de retención.

🎯 Objetivos

Realizar Análisis Exploratorio de Datos (EDA)

Identificar variables que influyen en el churn

Analizar patrones de comportamiento de clientes

Generar conclusiones basadas en datos

🛠 Herramientas Utilizadas

Python

Pandas

NumPy

Matplotlib / Seaborn

Jupyter Notebook

📊 Análisis Realizado

Limpieza y preparación de datos

Análisis de variables categóricas y numéricas

Distribución del churn

Análisis de correlaciones

Identificación de patrones de riesgo

📈 Principales Hallazgos

El tipo de contrato influye significativamente en la probabilidad de cancelación.

Clientes con menor antigüedad presentan mayor tasa de churn.

Los cargos mensuales tienen relación con la retención del cliente.
